# Table Order & Store Management System

飲食店向けの次世代テーブルオーダー・店舗管理システムです。お客様のモバイル注文から、厨房での調理管理、ホールでの配膳、レジでの精算、そして管理者による経営分析まで、店舗運営の全フローをカバーする統合ソリューションです。

## 🌟 主な機能

### 1. お客様用モバイル注文 (Menu & Details)
- **ページ遷移型の商品詳細**: モーダルから専用ページ遷移へ刷新。100文字の長文名や、詳細な商品説明、アレルギー情報を確実に閲覧可能。
- **注文履歴の確認 (History)**: カート内の合計だけでなく、これまでの注文履歴と総額をいつでも確認可能。
- **プレミアムな操作感**: 確定ボタンの質感、ホバー効果、視認性の高いレイアウトなど、高級感を追求。

### 2. 店舗運営支援
- **キッチンボード (Kitchen)**: 調理待ちアイテムを一覧表示。調理完了をワンタップでホールに通知。
- **配膳管理 (Hall)**: 配膳が必要なテーブルと商品をリアルタイムに把握。
- **レジ精算 (Cashier)**: テーブルごとの注文明細を自動集計し、精算処理を実行。

### 3. 管理者ダッシュボード (Admin)
- **フロア監視ボード**: 店舗全体の稼働状況をリアルタイムで俯瞰。
- **商品情報の完全な CRUD 管理**: メニュー名、価格、カテゴリーに加え「商品説明」「アレルギー情報」「販売状態」を自在に編集・更新可能。
- **売上統計分析**: 累計売上、日次トレンド、商品別ランキングの可視化。

## 🎨 洗練された UI システムと高度な CSS アーキテクチャ
全12画面にわたるインラインスタイルを廃止し、中央集約型のデザインシステムへと刷新しました。

- **中央集約型 CSS設計**: `style.css` への完全集約により、全画面で一貫したプレミアムなデザインと高い保守性を実現。
- **マルチカラム・レイアウト**: 2カラム（管理・運営）から3カラム（顧客用カート）まで、共通の `.app-layout` システムで柔軟に対応。
- **コンポーネントベースのデザイン**: カード、統計、一覧、モニターグリッドなどを共通クラス化し、新しい画面の構築も極めて容易に。
- **インテリジェント・テーマ**: キッチン画面専用のダークテーマをクラス一つの切り替えで提供し、現場の視認性を最大化。

## 🛡️ システムの堅牢性・安全性
商用レベルの運用に耐えうる、高度なセキュリティ基盤を構築しています。

- **多層防御のパスワード保護**: SHA-256 + 10,000回ストレッチング + **個別ソルト** + **共通ペッパー** を組み合わせ、DB流出時も極めて高い保護性能を発揮。
- **Webセキュリティ標準**: XSS, CSRF, セッション固定攻撃の防止策を網羅。
- **ロールベースアクセス制御**: `AuthFilter` による厳格な権限管理。

- **洗練されたデザインシステム**: CSS変数を駆使したダークモード対応。

## 🚀 技術スタック
- **Backend**: Java (Jakarta EE 10+), Servlet/JSP
- **Frontend**: Vanilla HTML5, Modern CSS (Variables/Grid/Flexbox/Centralized Architecture)
- **Asset**: AI-Generated Custom Favicon
- **Database**: MySQL / MariaDB (マルチ対応)
- **Server**: Apache Tomcat 11+

## 🛠️ セットアップ方法

1. **前提条件**: JDK 17+, Apache Tomcat 11+, MySQL
2. **データベース構築**: `src/main/webapp/sql/` 内の `schema.sql`, `seed.sql` 等を実行。
3. **マイグレーション**: 既存環境への適用は `sql/` 内の最新のマイグレーションファイルを順に適用。
4. **設定**: `database.properties` でDB接続情報と `app.security.pepper` を設定。
